# Orbital Bridge

## Run Tests

```shell
npx hardhat test
```

## Deploy to Testnet

```
PRIVATE_KEY= RPC_URL= npx hardhat --network sepolia deploy --tags deploy
```

## Deposit to Bridge

```
PRIVATE_KEY= RPC_URL= npx hardhat --network sepolia deploy --tags deposit
```

## Addresses

Addresses can be found in `deployments` directory.
